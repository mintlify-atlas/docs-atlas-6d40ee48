## Overview

GitWhisper is available as plugins for popular code editors, bringing AI-powered commit message generation directly into your development workflow. Instead of switching to the command line, you can generate commit messages right from your editor's Git interface.

## Benefits of Editor Integrations

- **Streamlined Workflow**: Generate commit messages without leaving your editor
- **Native Integration**: Works seamlessly with your editor's built-in Git tools
- **Consistent Experience**: Same AI-powered quality as the CLI tool
- **Time Savings**: Faster commit process with fewer context switches
