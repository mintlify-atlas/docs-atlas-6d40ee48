## Overview

The GitWhisper plugin for JetBrains IDEs provides seamless integration with JetBrains VCS tools, enabling AI-powered commit message generation across all JetBrains products.

## Supported IDEs

The GitWhisper plugin works with all JetBrains IDEs, including:

- IntelliJ IDEA
- WebStorm
- PhpStorm
- PyCharm
- GoLand
- RubyMine
- CLion
- Rider
- DataGrip
- Android Studio

### Alternative Installation Methods

1. **From JetBrains IDE**:
   - Open your JetBrains IDE
   - Go to `File` → `Settings` → `Plugins` (Windows/Linux)
   - Or `IntelliJ IDEA` → `Preferences` → `Plugins` (Mac)
   - Click on the `Marketplace` tab
   - Search for "GitWhisper"
   - Click Install
   - Restart your IDE

## Features

- **VCS Integration**: Seamlessly integrates with JetBrains VCS tools
- **AI-Powered Commit Messages**: Automatically generates meaningful commit messages based on your changes
- **Conventional Format**: Follows conventional commit format with emojis
- **Cross-IDE Support**: Works consistently across all JetBrains products

## Usage

Once installed, the plugin integrates with your JetBrains IDE's VCS system:

1. Stage your changes in the Commit tool window
2. Use the GitWhisper action to generate a commit message
3. Review and modify the generated message as needed
4. Commit your changes
