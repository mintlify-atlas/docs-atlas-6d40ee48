GitWhisper supports multiple AI providers and models. Use these commands to discover available options.

## list-models

Display all supported AI model providers.

### Usage

```bash
gitwhisper list-models
# or use the shorthand
gw list-models
```

### Terminal Output

```bash
$ gw list-models
Available models:
  - claude (Anthropic Claude)
  - openai (OpenAI GPT models)
  - gemini (Google Gemini)
  - grok (xAI Grok)
  - llama (Meta Llama)
  - deepseek (DeepSeek, Inc.)
  - github (GitHub)
  - ollama (Ollama)
  - free (LLM7.io - No API key required)
```

## list-variants

Display available variants for AI models.

### Usage

```bash
gw list-variants
```

List all variants for all models.

```bash
gw list-variants --model <model-name>
```

List variants for a specific model.

### Options

<ParamField path="--model" type="string">
  Filter variants by model.
  
  **Allowed values:** `claude`, `openai`, `gemini`, `grok`, `llama`, `deepseek`, `github`, `ollama`, `free`
  
  **Shorthand:** `-m`
</ParamField>

### Examples

<CodeGroup>
```bash All variants
gw list-variants
```

```bash OpenAI variants
gw list-variants --model openai
```

```bash Claude variants
gw list-variants -m claude
```
</CodeGroup>

## Using Models and Variants

### In Commands

```bash
# Use specific model
gw commit --model claude

# Use specific variant
gw commit --model openai --model-variant gpt-4o

# Analyze with Gemini Pro
gw analyze --model gemini --model-variant gemini-2.5-pro
```

### Set as Defaults

```bash
# Set default model
gw set-defaults --model claude

# Set default model and variant
gw set-defaults --model openai --model-variant gpt-4o
```

### Model Selection Guide

**For best quality:**
- `claude` with `claude-opus-4-1-20250805`
- `openai` with `gpt-5` or `o1-preview`
- `gemini` with `gemini-2.5-pro`

**For best speed:**
- `openai` with `gpt-4o-mini`
- `claude` with `claude-3-haiku-20240307`
- `gemini` with `gemini-2.5-flash-lite`

**For cost efficiency:**
- `deepseek` with `deepseek-chat`
- `gemini` with `gemini-2.5-flash-lite`
- `free` (no API key required)

**For local/offline:**
- `ollama` with any local model

**For coding tasks:**
- `grok` with `grok-code-fast-1`
- `deepseek` with `deepseek-reasoner`
- `github` with `Codestral 25.01`
